쉐어하우스 홈페이지: ShareHouse

소개

이 프로젝트는 스프링 MVC 구조를 학습하며 웹의 전체적인 로직을 그려볼수있었던 프로젝트였습니다. Oracle DB연동을 통해 게시판, 댓글을 구현하였으며 회원가입 및 로그인 예약시스템을 구현해보았습니다.

Project information

  • title: ShareHouse
  • Category: Web
  • personnel: 3명
  • 개발 기간: 2020.08 (약2주)
  • Github: https://github.com/rosearemi/jsp_project_ShareHouse.git

Project 개발환경

  • DBMS: Oracle
  • Language: JAVA
  • Framework: Spring
  • IDE: STS

주요기능

  • 회원가입/로그인
  • 위치 소개: 구글 지도 api사용
  • 게시판: 게시판 작성 및 댓글, 게시글 상세보기
  • 예약: 원하는 날짜에 예약
