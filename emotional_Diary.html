내 감정을 분석해주는 다이어리 : Emotional_Diary

제작 배경

장기화된 코로나로 인해 자신의 감정을 되돌아 보지 못한 채 우울하게 넘기는 날들이 많아지면서 이러한 현상을 일컫는 '코로나 블루' 라는 신조어도 생겼습니다. 이 프로젝트는 이러한 현상에서 자신의 감정을 저장하고 공유할 수 있는 서비스를 제공하여 자신의 감정을 한번 더 되돌아 보거나 서로의 감정을 공감하고 이해할 수 있게 감정분석을 해주는 다이어리입니다.

Project information

  • title: Emotional Diary
  • Category: Web&App
  • personnel: 5명
  • 개발 기간: 2020.11.30~2020.12.15
  • Github: https://github.com/rosearemi/Emotional-Diary.git

Project 개발환경

  • DBMS: MySQL
  • Language: JAVA
  • Framework: SpringBoot, Django, React, Android
  • IDE: STS, VS Code, Android Studio
  • 형상관리: Github

주요기능

  • 회원가입/로그인
  • 그룹생성: 일기를 공유할 그룹생성 (그룹 생성, 초대, 탈퇴)
  • 글 작성: 글목록,글수정,글삭제
  • 감정분석: 작성글의 내용 형태소 단위로 분류, 분석한 정보와 함께 글저장
